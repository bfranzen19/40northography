triggeraware.com - this database allows you to search by tv/movie title or trigger type to ascertain whether or not there is a risk for you or another survivor to be triggered. users can also fill out a form to recommend triggering scenes to add to the database. this application was built as my final project at CodeCraft and was built using web scraping (cheerio and request), javascript, mongoDB (using mongoose to validate data), express, fuze.js (fuzzy search JS library), bootstrap 4, CSS, HTML, bcrypt for authentication to login and verify the recommended scenes from users.
